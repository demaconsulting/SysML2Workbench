# Introduction

This document collection provides release-oriented build notes for SysML2Workbench,

along with the baseline local toolchain guidance needed to build and validate the

application during the current phase-0 desktop-viewer effort.

## Purpose

The build notes provide auditable build and release context for SysML2Workbench.

They combine checked-in guidance for local developer setup with generated records of

change history and tool versions so that a release can be traced back to the

environment that produced it.

## Scope

This collection covers:

- The expected local .NET and Avalonia-oriented development toolchain for this repository

- Generated release notes, version history, and tool-version evidence produced by the compliance pipeline

- The current phase-0 scope of SysML2Workbench as a read-only desktop viewer and custom view builder
