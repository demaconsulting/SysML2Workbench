# Introduction

This document records code-quality evidence for SysML2Workbench, the cross-platform

Avalonia desktop viewer for SysML v2 textual models.

## Purpose

The purpose of this collection is to preserve static-analysis and quality-checking

evidence for the repository so that findings can be reviewed, tracked, and resolved

as the application evolves from a phase-0 read-only viewer into a fuller IDE.

## Scope

This collection covers generated or imported quality evidence associated with the

repository source base, including static analysis, formatting compliance, and

related quality checks applied to the implementation under `src/`.
